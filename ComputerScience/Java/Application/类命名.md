- entity：表示实体类，与 model 包名类似，常用于表示数据库中的表或视图对应的实体类。
- domain：表示领域对象，常用于表示业务领域中的对象，例如订单、商品、用户等。
- po 或 pojo：表示持久化对象或普通 Java 对象，常用于表示与数据库交互的对象。
- bean：表示 JavaBean，通常用于表示符合 JavaBean 规范的对象。
- dto：表示数据传输对象，常用于表示应用程序中不同层之间传输的数据对象。
- vo：表示值对象，常用于表示不可变对象，例如枚举类型、常量等。
- model ：表示实体对象，与 entity包名类似，常用于表示领域模型中的实体对象。
