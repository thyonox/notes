# 计算机系统体系结构

二进制中使用补码表示负数：

1. 正数的补码与原码相同
2. 复数的补码是其对应正数的源码取反，然后加1
3. 最高位是符号位

例如：

5的二进制是：0101

-5则为：先取反1010，加1则为1011

负数的补码是不能直接将最高位当作符号位然后其余为转为十进制得到负数的

要想得知十进制，则需要再次进行相同的计算，求得它的正数

对操作数进行取反操作：

1. 正数取反为负数，负数取反为正数
2. 操作数绝对值减1

### 异或运算的基本规律

1. **相同位异或为 0，不同位异或为 1**：
    - 异或运算对每一位进行比较，如果两个位相同则结果为 `0`，如果不同则结果为 `1`。
    - 例如：`1 ^ 1 = 0`, `0 ^ 0 = 0`, `1 ^ 0 = 1`, `0 ^ 1 = 1`。
2. **自反性（对称性）**：
    - `A ^ B` 与 `B ^ A` 的结果相同，即异或运算是对称的。
    - 例如：`5 ^ 3` 等于 `3 ^ 5`。
3. **结合性**：
    - `(A ^ B) ^ C` 等于 `A ^ (B ^ C)`，这表示异或运算可以按任意顺序结合而不改变结果。
4. **与 0 的异或**：
    - 任何数与 `0` 异或结果为该数本身。
    - `A ^ 0 = A`。
5. **与自身的异或**：
    - 任何数与自身异或结果为 `0`。
    - `A ^ A = 0`。

### 异或运算的特性和应用

1. **交换两个变量的值（不使用临时变量）**：
    - 异或运算可以用于交换两个变量的值而无需使用额外的空间。假设有两个变量 `a` 和 `b`：
        
        ```java
        javaCopy code
        a = a ^ b;
        b = a ^ b; // 现在 b = (a ^ b) ^ b = a
        a = a ^ b; // 现在 a = (a ^ b) ^ a = b
        
        ```
        
    - 这样经过三次异或运算后，`a` 和 `b` 的值被交换了。
        
2. **找出数组中唯一的元素**：
    - 给定一个数组，其中所有的元素都成对出现，只有一个元素是单独的。异或所有的元素，成对出现的元素会互相抵消（因为 `A ^ A = 0`），最后得到的结果就是唯一的那个元素。
    - 例如：`[2, 3, 2, 4, 4]`，执行 `2 ^ 3 ^ 2 ^ 4 ^ 4` 结果为 `3`。
3. **用于生成伪随机数**：
    - 异或运算在生成伪随机数的算法中也被使用，因为它可以打乱位的分布，增加不确定性。
